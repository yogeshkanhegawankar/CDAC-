---note
if there are multiple classes in the file then only 1 class will be defined as public, 
the class which has the main method will be a public class.
the other class should have the less accessibility. or
all the classes can have the same accessiblity (default)
a class  can not be private

if no access specifie is written , then it is default for class, mf, mv
if there is a single class , the class be public or default


--access specifiers / access modifiers
1 private		: mv			: it is accessible to the same class
2 protected	: mv, mf		:  it is accessible to the child class only
3 default		: class, mv, mf :  it is accessible to the other classes in the same package
4 public		: class , mf  	: it is accessible to the all classes inside and outside the 			package
